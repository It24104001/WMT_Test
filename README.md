# Item Manager MERN Application
This project is an Item Manager application enhanced with a "Temperature Requirement" field for a MERN Lab Test.

## Features
- Full CRUD functionality for Items.
- New Field: Temperature Requirement.
- Responsive UI built with React and Vite.
- Backend API built with Node.js, Express, and MongoDB.

## Project Structure
- `backend/`: Node.js Express API.
- `frontend/`: React Vite application.

## Local Setup
1. **Backend**:
   - Navigate to `backend/`.
   - Run `npm install`.
   - Create a `.env` file with `MONGO_URI` and `PORT`.
   - Run `npm run dev`.
2. **Frontend**:
   - Navigate to `frontend/`.
   - Run `npm install`.
   - Create a `.env` file with `VITE_API_URL`.
   - Run `npm run dev`.

## Deployment
- **Backend**: Deployed on Render.
- **Frontend**: Deployed on Vercel.
